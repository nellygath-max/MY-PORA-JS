Fetch One Country

The simplest possible example — get data for Nigeria

Example 1 
// Step 1 — Ask the API for Nigeria's data
fetch('https://restcountries.com/v3.1/name/nigeria')

  // Step 2 — Convert the response to JSON
  .then(response => response.json())

  // Step 3 — Use the data
  .then(data => {
    const country = data[0];   // API returns an array, take first item
    console.log(country.name.common);   // "Nigeria"
    console.log(country.capital[0]);     // "Abuja"
    console.log(country.population);    // 218541212
  });
💬 What each line does
→
fetch(url) — sends a request to the API and gets country data back
→
.then(response => response.json()) — the API sends raw text, .json() converts it into a JavaScript object you can read
→
data[0] — the API wraps results in an array [ ], so we take the first item with [0]
→
country.name.common — reads the name. country.capital[0] reads the capital city. country.population reads the number

Show 5 Countries as Cards

Fetch all countries, pick 5, display them on the page

Example 2 
// HTML you need: <div id="cards"></div>

async function showFiveCountries() {

  // 1. Fetch ALL countries
  const response = await fetch('https://restcountries.com/v3.1/all');
  const all      = await response.json();

  // 2. Take only the first 5
  const five = all.slice(0, 5);

  // 3. Build HTML for each country and put it on the page
  const container = document.getElementById('cards');

  five.forEach(country => {
    container.innerHTML += `
      <div style="border:1px solid #333; padding:12px; margin:8px">
        <img src="${country.flags.svg}" width="60"/>
        <h3>${country.name.common}</h3>
        <p>Capital: ${country.capital?.[0] || 'N/A'}</p>
        <p>Population: ${country.population.toLocaleString()}</p>
      </div>
    `;
  });
}

showFiveCountries();
💬 What each line does
→
async / await — cleaner way to wait for the API response. Code reads top-to-bottom like normal
→
.slice(0, 5) — cuts the array. slice(0, 5) means "give me items from position 0 up to (not including) 5" = first 5 items
→
.forEach(country => { }) — loops through each of the 5 countries one by one
→
container.innerHTML += — adds a new card to the page each time the loop runs
→
country.capital?.[0] — the ?. means "only read [0] if capital exists". Some countries have no capital so this prevents a crash
→
.toLocaleString() — formats 218541212 into "218,541,212" with commas for readability

Search for Any Country

Type a name, press Search, see the result

Example 3 
/* HTML you need:
   <input id="search-box" placeholder="Type a country name"/>
   <button onclick="searchCountry()">Search</button>
   <div id="result"></div>
*/

async function searchCountry() {

  // 1. Read what the user typed
  const name = document.getElementById('search-box').value;

  // 2. Fetch that country from the API
  const response = await fetch(
    `https://restcountries.com/v3.1/name/${name}`
  );

  // 3. If not found, show an error message
  if (!response.ok) {
    document.getElementById('result').innerHTML =
      '<p>Country not found. Try again.</p>';
    return;
  }

  // 4. Parse and show the result
  const data    = await response.json();
  const country = data[0];

  document.getElementById('result').innerHTML = `
    <img src="${country.flags.svg}" width="80"/>
    <h3>${country.name.common}</h3>
    <p>Capital: ${country.capital?.[0]}</p>
    <p>Population: ${country.population.toLocaleString()}</p>
    <p>Region: ${country.region}</p>
  `;
}
💬 What each line does
→
document.getElementById('search-box').value — reads whatever text the user typed into the input box
→
fetch(`.../${name}`) — we put the user's typed name directly in the URL using ${ }. So typing "Ghana" fetches /name/ghana
→
if (!response.ok) — if the country doesn't exist, the API returns a 404 error. We check this and show a friendly message instead of crashing
→
innerHTML = `...` — replaces whatever is inside the result div with the new country info
